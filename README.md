# Abandoned Camp Port

Hello! This is a small datapack focused on adding the Abandoned Camp from 26.3 snapshots into older versions. Note that this is based on 26.3 Snapshot 1, so the structure is subject to change.


<details>
<summary>Technical Details</summary>

Version Backporting:
- 25w41a-26.2: No change
- 25w31a-1.21.10: Removed spears
- 24w21a-1.21.8: Changed Copper tools into Stone and Copper armor into Leather (make an issue on GitHub if you feel it should be a different material)
- 1.18.2-24w20a (`data` folder): Changed loot table format to match old format

</details>



## Disclaimer

This datapack is set to work for `1.18.2` and above. Please make an issue in my GitHub if it works on older versions or doesn't on `1.18.2` or any version above.
